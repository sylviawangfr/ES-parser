Error event IDs and error codes

Error codes describe a service procedure that must be followed. Each event ID that requires service has an associated error code.

Error codes can be either notification type E (error) or notification type W (warning). Table 1 lists the event IDs and corresponding error codes, the notification type, and the condition of the event.
