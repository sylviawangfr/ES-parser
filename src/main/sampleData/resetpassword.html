resetpassword

 

Name

resetpassword

Use the resetpassword command to reset the clustered system (system) superuser password to passw0rd.

 

Syntax


>>- satask -- resetpassword -- --------------------------------><



      

Parameters

satask

         System Administrator task; service commands that are
         only used in specific circumstances.  

Description

This command resets the system superuser password to passw0rd. You are prompted for a new password when you next log in to the graphical user interface (GUI).  

An invocation example

An invocation example

satask resetpassword

The resulting output:

No feedback
